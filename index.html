What is the difference between var, let, and const

What is the difference between map(), forEach(), and filter()
map() which loops over an array returns an array. forEach() which loops over an array returns nothing. filter() which loops over an array returns all values ​​in the array if the condition is true.
What are arrow functions and how they are different from regular functions
In JavaScript (ES6), an arrow function can automatically return a value if you do not use curly braces {}.But a normal function does not automatically return a value — you must explicitly use the return keyword
How do JavaScript Promises work
When an API call is made, the server first sends a response. If the request is successful (resolved), it eturns the data in packets. Then we use .then() to handle the response and convert it into JSON using response.json(). After the JSON onversion is resolved, we get the actual usable data. If something goes wrong (rejected), the error is caught in the .catch() block.
How does closures work in JavaScript
A closure is a feature in JavaScript where an inner function can remember and access the variables of its outer function, even after the outer function has finished executing.
